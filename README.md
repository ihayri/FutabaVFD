# FutabaVFD
FutabaVFD is a lightweight, hardware‑agnostic Arduino library for driving Futaba 6‑digit VFD modules using a simple 3‑wire serial interface (DIN, CLK, CS) plus RESET. It supports both classic AVR boards (Arduino UNO, Nano, Pro Mini) and modern high‑speed MCUs like the XIAO ESP32‑S3, thanks to a carefully tuned bit‑banged protocol.
